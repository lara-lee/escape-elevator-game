퇴근 엘리베이터 Escape Elevator

사용자 흐름v2.02026-06-19
기획안 기능 명세서 사용자 흐름

빌딩 꼭대기에서 시작해 빈 틈에 문을 닫아 한 층씩 내려가고, 1층 로비에 도착하면 퇴근 성공. 하트 3개를 다 잃으면 야근(게임 오버).

핵심 루프
문 닫기 → 2초 무방해 시 한 층 하강(점수+100) / 닫는 중 승객 도착 시 실패(하트 −1) → 반복 → 1층 도착(승리) 또는 하트 0(패배).
메인 루프
단계 선택 → 3·2·1·START 카운트다운
엘리베이터 대기(문 열림) · 승객 우측에서 접근
문 닫기 (버튼 / Space · ↓ · Enter)
2초 카운트 — 승객 도착 여부 판정
성공 → 한 층 ▼ 하강 · 점수 +100 · 콤보 · 상승(하강) 연출
실패(충돌) → 문 재개방 · 하트 −1 · 콤보 초기화 (택배=지연 / 부장·대표=강제)
1층 로비 도착 → 🎉 퇴근 성공!
하트 0 → 🫠 게임 오버(야근)
결과 화면 → 다음 단계 / 다시하기 / 메인

플레이어 관점 시나리오와 예외.

S1. 단계 선택 ~ 첫 하강
메인 → 게임 시작 → 단계 선택(1~5) 또는 연습
카운트다운(3·2·1·START) 동안 대기
빈 틈에 문 닫기 → 2초 유지
첫 출발 성공 → 한 층 하강
S2. 연습 게임 (위험 미리보기)
메인 → 연습 게임 (6층 · 하트 5)
지금 닫으면 늦는 승객 = 빨간 펄스 + ! 표시
위험 표시 없을 때 닫기 연습 → 타이밍 학습
S3. 실패 (승객 충돌)
문 닫는 중 승객 도착
문 재개방 · 하트 −1 · 흔들림/진동
체크포인트마다 ❤️ 하트 낙하 → 탭해서 잡으면 +1
택배 충돌 → 잠깐 닫기 잠금(지연)
부장 / 대표 충돌 → 강제 개방, 대표는 초고속이라 가장 위험(5단계)
S4. 클리어 ~ 다음 단계
1층 도착 → 퇴근 성공! · 단계별 최고 점수 저장
결과 → "다음 단계 ▶" (5단계까지) / "다시하기" / "메인"
엣지 케이스
연타 닫는 중 추가 입력 무시(중복 타이머 방지)
일시정지 ⏸ / Esc · P → 갱신·입력 정지, 오버레이
백그라운드 탭 비활성 시 delta time 클램프로 폭주 방지
저장소 차단 LocalStorage 불가 시 메모리 폴백(최고 점수 비영구)
사운드 mp3 없거나 음소거면 무음, 효과음·배경음 개별 토글

4개 화면 전환: 메인 → 단계 선택 → 게임 → 결과.

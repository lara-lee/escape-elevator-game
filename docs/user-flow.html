퇴근 엘리베이터 Escape Elevator

사용자 흐름 & 기능 구성 v1.0 2026-06-19 캐주얼 웹게임 · 1차 Web → 2차 PWA
기획안 기능 명세서 사용자 흐름

엘리베이터를 출발시켜 퇴근하려 하지만 승객들이 끊임없이 방해한다. 스트레스를 관리하며 최대한 많은 층을 올라가는 것이 목표인 엔드리스 루프 게임.

핵심 루프 한 줄 요약

문닫기 → 3초 무방해 시 출발 성공(층수↑·점수↑·스트레스↓) / 승객 충돌 시 출발 실패(문 재개방·스트레스↑) → 반복 → 스트레스 100% 시 게임 오버.

플레이어 관점의 주요 시나리오와 분기, 예외(엣지 케이스)를 정리한다.

단일 페이지 내 3개 화면 전환 구조 (메인 → 게임 → 결과 → 메인/게임).

화면 맵

① 메인 화면
· 게임 제목
· 시작 버튼
· 최고 점수 표시 (F-012)
② 게임 화면
· 현재 층수 / 점수
· 스트레스 게이지
· 엘리베이터 + 승객 영역
· 문닫기 버튼
③ 결과 화면
· 최종 점수 / 도달 층수
· 게임 통계
· 다시하기 버튼

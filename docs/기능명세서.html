상세 명세

F-001 게임 시작

  • 단계 선택(1~5) 또는 연습 게임 선택 → 상태 초기화(층수=빌딩 꼭대기, 하트=최대, 점수 0).
  • 3 · 2 · 1 · START 카운트다운 동안 생성/이동/입력 정지(introTimer).

F-002 문 닫기

  • 입력: 문닫기 버튼(pointerdown) / Space · ↓ · Enter.
  • 조건: 문 열림 + 카운트다운 종료 + 지연/일시정지 아님 → 문 닫힘 시작(2초 타이머).

F-003 출발 판정 → 하강

  • 2초 동안 승객 도착 없으면 성공 → 현재 층 −1(하강), 점수·콤보, 출발 연출.
  • 1층 도달 시 F-010(승리).

F-004 승객 생성

  • 단계별 생성 주기(랜덤), 진행도에 따라 짧아짐.
  • 확률 로스터로 타입 결정. 대표(ceo)는 stage.extra에 포함된 단계(5단계)에서만 등장.

F-005 승객 이동

  • 매 프레임 좌측(엘리베이터)으로 이동(속도 × 단계배수 × 진행도가속, delta time).
  • 입구 도착 시: 닫는 중이면 F-006, 아니면 탑승 후 제거(무해).

F-006 승객 충돌 → 하트 −1

  • 문 재개방, 콤보 초기화, 하트 −1, 택배는 지연(닫기 잠금) 부여.
  • 하트 0 → F-010(실패). 진동(부장/대표는 강한 패턴).

F-007 하트(생명) 시스템

  • 최대 하트 = 단계 lives (1~3단계 3 / 4단계 4 / 5단계 5 / 연습 5). 0이면 게임 오버.
  • 보너스 하트(낙하·잡기): 균등 체크포인트마다 ❤️ 하트가 하늘에서 떨어짐 → 탭/터치로 잡으면 +1(최대치까지, 가득이면 +50점). 못 잡으면 바닥에서 소멸.
  • 등장 횟수 = 단계 번호 (1~5).

F-008 점수 · 콤보

  • 성공 +100, 콤보 보너스 +30×(연속−1). 실패 시 콤보 0.

F-009 층수(하강) · 진행도

  • floor는 꼭대기에서 1로 감소. 진행도 = 내려간 층수(successes).
  • 생성 주기·속도 모두 진행도에 비례해 난이도 상승.

F-010 게임 종료

  • 승리: 1층 도착(won=true) → 결과 "퇴근 성공". 실패: 하트 0 → "퇴근 실패(야근)".
  • 단계별 최고 점수 갱신 저장(F-012, 연습 제외). 사운드: 승리=victory / 실패=youlose.

F-011 다시하기 / 다음 단계

  • 다시하기: 같은 단계 재시작. 다음 단계: 승리 시 다음 단계(5단계까지) / 연습은 단계 선택으로.

F-012 단계별 최고 점수

  • ee_best_{stageId} 키로 LocalStorage 저장(미지원 시 메모리 폴백). 연습은 저장 안 함.

F-013 일시정지

  • ⏸ 버튼 / Esc · P 토글. 일시정지 중 루프 갱신·입력 정지, 오버레이(계속/메인).

F-014 사운드

  • 우상단 ⚙ 설정 → 배경음(🎵)/효과음(🔊) 각각 ON·OFF, 설정 LocalStorage 저장. sounds/ mp3 매핑.

F-015 연습 모드 + 위험 미리보기

  • 6층·하트 5·느린 설정. 지금 닫으면 2초 안에 도착할 승객을 빨간 펄스 + ! 로 표시.

F-016 제한 시간

  • 단계별 제한 시간(1단계 120s ~ 5단계 210s, 연습 무제한). 매 프레임 감소, 0이 되면 시간 초과 → 게임 오버(결과 "시간 초과").
  • HUD에 시간(m:ss) 표시, 15초 이하 빨강·깜빡임. 일시정지·시작 카운트다운 중에는 멈춤.

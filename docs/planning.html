2. 게임 컨셉

퇴근하려고 엘리베이터를 타지만, 문이 닫히기 직전 사람들이 뛰어들어 방해한다. 빈 틈을 노려 문을 닫고 한 층씩 내려가 1층 로비에 도착하면 퇴근 성공.

3. 게임 목표

  • 승리 조건: 1층(로비) 도착 → 퇴근 성공
  • 패배 조건: 하트 3개 모두 소진 → 야근(게임 오버)
  • 빌딩 꼭대기 층에서 시작해 1층까지 하강한다.

4. 게임 흐름

단계 선택
↓
3 · 2 · 1 · START (시작 카운트다운)
↓
엘리베이터 대기(문 열림)  ← 승객들이 우측에서 접근
↓
문 닫기 (버튼/Space/↓/Enter)
↓
3초 무방해 유지?
  ├─ 성공 → 한 층 하강 · 점수 +100 · (콤보)
  └─ 실패(승객 충돌) → 문 재개방 · 하트 −1
↓
1층 로비 도착 → 퇴근 성공! / 하트 0 → 게임 오버

5. 단계

모든 단계 20층 통일. 난이도(생성 주기·속도)로 차등하고, 어려운 단계일수록 하트 회복을 더 준다. 진행도(내려간 층수)에 따라 추가 가속.

단계 빌딩 생성 주기(초) 속도 배수 보너스 하트 비고
1단계 느긋한 오후 20층 2.3~3.7 0.85 ❤️×1 드뭄·느림
2단계 평범한 퇴근 20층 1.7~3.0 1.05 ❤️×2 표준
3단계 러시아워 20층 1.2~2.3 1.30 ❤️×3 자주·빠르게
4단계 야근 지옥 20층 0.85~1.7 1.60 ❤️×4 쉴 틈 없음
5단계 대표님 등장 20층 0.7~1.4 1.90 ❤️×5 대표 출현
연습 게임 6층 2.8~4.4 0.80 - 위험 미리보기 ON, 하트 5

6. 승객

승객 등장 확률 속도 특수효과 충돌 시
🎓 인턴 20% 느림(0.13) - 하트 −1
🧑‍💼 사원 60% 보통(0.16) - 하트 −1
📦 택배 10% 빠름(0.20) 닫기 잠금(지연 1.5초) 하트 −1
👔 부장 10% 매우 빠름(0.25) 강제 개방 하트 −1
🕴️ 대표 5단계 한정(약 13%) 초고속(0.34) 강제 개방 하트 −1

하트 소모는 전부 −1로 동일. 차이는 속도·특수효과·등장 단계. 문이 열려 있을 때 도착한 승객은 그냥 타고 사라져 무해(닫는 중 도착만 실패).

7. 하트(생명) 시스템

  • 시작 하트 3개(4·5단계는 4·5개, 연습 5개). 충돌 1회당 −1, 0개 → 게임 오버.
  • 보너스 하트: 단계 체크포인트마다 ❤️ 하트가 하늘에서 낙하 → 탭/터치로 잡으면 +1(최대치까지, 가득이면 +50점).
  • 등장 횟수 = 단계 번호 (1단계 1개 … 5단계 5개). 못 잡으면 바닥에 떨어져 사라짐(이월 없음).

8. 점수 · 콤보

  • 출발 성공 시 +100, 연속 성공마다 콤보 보너스 +30.
  • 실패하면 콤보 초기화.
  • 단계별 최고 점수를 LocalStorage에 저장(연습은 저장 안 함).

11. 기술 · 설계 원칙

  • HTML5 / CSS3 / Vanilla JS, DOM + CSS 렌더링.
  • 로직/화면 분리(game.js ↔ ui.js), 저장소 추상화(storage.js).
  • pointer 입력(터치+마우스) + 키보드(Space/↓/Enter, Esc/P).
  • requestAnimationFrame + delta time 루프(백그라운드 dt 클램프).
  • 난이도는 진행도(내려간 층수) 기준으로 가속.

12. PWA

  • manifest.webmanifest + Service Worker(sw.js, network-first → 온라인 최신 / 오프라인 캐시).
  • 설치 아이콘 192/512, apple-touch, favicon. localhost·HTTPS에서 설치/오프라인.

13. 사운드

  • sounds/ mp3 매핑(js/audio.js). 효과음 🔊 / 배경음 🎵 각각 토글(설정 저장).
